# Pattern · Provenance & disclosure

> Tray 3 · **Governance pattern.** Ensures AI involvement and sources are visible everywhere AI output appears.

## Problem it solves

Ensures AI involvement and sources are visible everywhere AI output appears.

## Approach

- Mark AI-generated/assisted content at the point of display.
- Attach sources to factual claims.
- Show which model produced the output.

## Built from

- DisclosureBadge
- CitationChip
- SourcesPanel
- ModelSelector

## Do / Don't

**Do:** Disclose AI involvement visibly (not buried); Cite sources for factual claims.

**Don't:** Let users mistake generated content for verified fact; Hide the producing model.
