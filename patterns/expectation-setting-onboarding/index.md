# Pattern · Expectation-setting onboarding

> Tray 3 · **Agentic pattern.** Teaches that AI output is non-deterministic and fallible, up front, so trust is calibrated.

## Problem it solves

Teaches that AI output is non-deterministic and fallible, up front, so trust is calibrated.

## Approach

- On first use, set expectations: capabilities, limits, fallibility.
- Pair with disclosure + starter prompts.
- Re-surface limits contextually, not just once.

## Built from

- DisclosureBadge
- PromptSuggestions
- Alert
- RefusalState

## Do / Don't

**Do:** State capabilities AND limits early; Use starter prompts to show what is possible.

**Don't:** Overpromise capability; Bury the "can be wrong" disclosure in settings.
